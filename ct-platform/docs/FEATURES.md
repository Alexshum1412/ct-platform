## ✅ Реализованные функции

### 1. 🎨 UI/UX

#### Постоянная шапка навигации
- **Логотип** в левом верхнем углу с быстрым переходом на главную
- **Навигация по предметам** — быстрый доступ к 10+ предметам
- **Индикатор онлайн-пользователей** — фейковый счётчик, зависящий от времени суток
- **Переключатель темы** — светлая/тёмная тема
- **Меню профиля** — выпадающее меню с быстрыми действиями

#### Тёмная тема
- Полная поддержка `dark mode`
- Автоматическое определение системной темы
- Плавные переходы между темами
- Сохранение выбора в localStorage

### 2. 🔐 Авторизация

#### Страницы входа и регистрации
- Валидация форм в реальном времени

- OAuth (Google, Telegram) — UI готов
- Автоматический редирект после входа

#### Профиль пользователя
- Редактирование данных
- Загрузка аватара
- Информация о городе и школе
- Статистика по предметам

### 3. 📚 Раздел теории

#### Структура
- Навигация по темам и подтемам
- Поиск по теории
- Связь с практическими заданиями

#### Формулы
- Красивое отображение формул

#### Примеры
- Разворачиваемые решения
- Пошаговые объяснения
- Практика по теме

---

### 4. 🎯 Подсказки (Хинты)

#### Три уровня подсказок
1. **Маленькая подсказка** — намёк на подход к решению
2. **Подробная подсказка** — объяснение метода
3. **Пошаговое решение** — детальное руководство

#### Особенности
- Не требует ответа для просмотра
- Помечает использованные подсказки
- Не влияет на статистику

---

### 5. 🎬 Видео-разборы

#### Интеграция в карточку задания
- Блок с информацией о видео
- Длительность и тема
- Кнопка просмотра

#### Типы видео
- Разбор типичных ошибок
- Объяснение сложных тем
- Советы по решению

---

### 6. 🏆 Геймификация

#### Система уровней
- XP за решение заданий
- Повышение уровня при накоплении XP
- Уведомления о повышении

#### Достижения
- 15+ достижений в разных категориях
- Редкость: Обычное, Редкое, Эпическое, Легендарное
- XP за разблокировку
- Страница всех достижений

#### Streak (Серия)
- Отслеживание дней подряд
- Бонусы за длинные серии
- Напоминания о тренировке

#### Бейджи
- Визуальные награды
- Отображение в профиле
