# 🎨 Дизайн-система CT-Platform

## Философия дизайна

### Принципы
1. **Спокойствие** — нейтральные цвета, ничего не отвлекает от учёбы
2. **Ясность** — всё должно быть понятно с первого взгляда
3. **Мотивация** — прогресс виден, достижения поощряются
4. **Доступность** — работает на любом устройстве

## Адаптивность

### Breakpoints
```css
/* Мобильные */
--screen-sm: 640px;

/* Планшеты */
--screen-md: 768px;

/* Ноутбуки */
--screen-lg: 1024px;

/* Десктоп */
--screen-xl: 1280px;

/* Большие экраны */
--screen-2xl: 1536px;
```

### Поведение

#### Мобильные (< 768px)
- Одноколоночный layout
- Нижняя навигация
- Полноэкранные карточки
- Упрощённая статистика

#### Планшеты (768px - 1024px)
- Двухколоночный layout
- Боковая навигация (сворачивается)
- Карточки 2 в ряд

#### Десктоп (> 1024px)
- Трёхколоночный layout
- Постоянная боковая навигация
- Карточки 3-4 в ряд
- Расширенная статистика
