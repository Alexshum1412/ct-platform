## 📋 Описание

CT-Platform — это современная EdTech-платформа для подготовки к централизованному тестированию (ЦТ) и централизованным экзаменам (ЦЭ) в Республике Беларусь.

### ✨ Основные возможности

- 📚 **10+ предметов** — Математика, Русский язык, Физика, Химия, Биология, История, Английский и др.
- 🎯 **Режим практики** — Задания по типам с фильтрами по темам и сложности
- ⏱️ **Режим экзамена** — Полноценный пробный экзамен с таймером
- 📖 **Подробные объяснения** — Каждое задание содержит решение и пояснение
- 💡 **Подсказки** — Три уровня подсказок (маленькая, подробная, пошаговая)
- 🎬 **Видео-разборы** — Уроки по типичным ошибкам
- 📊 **Статистика прогресса** — Отслеживание результатов и точности
- 🏆 **Геймификация** — XP, уровни, достижения, streak
- 🏅 **Лидерборд** — Соревнование с другими учениками
- ⭐ **Избранное** — Сохранение сложных заданий для повторения
- 🌙 **Тёмная тема** — Поддержка светлой и тёмной темы
- 🛠 **Админ-панель** — Управление контентом

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- npm 9+
- PostgreSQL 14+ (для backend)
- Redis 7+ (для backend)

### Установка

```bash
# 1. Клонировать репозиторий
git clone https://github.com/yourusername/ct-platform.git
cd ct-platform

# 2. Установить frontend
cd app
npm install

# 3. Установить backend
cd ../backend
npm install

# 4. Настроить окружение
cp .env.example .env
# Отредактировать .env

# 5. Настроить базу данных
npx prisma migrate dev
npx prisma generate

# 6. Запустить dev-серверы
# Терминал 1 (backend):
npm run dev

# Терминал 2 (frontend):
cd ../app
npm run dev
```

Откройте http://localhost:5173

## 🎯 Функциональность

### Главная страница
- Выбор предмета из списка
- Статистика платформы
- Описание возможностей

### Страница предмета
- Информация о предмете
- Быстрые действия (практика/экзамен)
- Список тем
- Недавние задания

- Фильтрация по темам

- Навигация по заданиям

- Подсказки трёх уровней

- Таймер с обратным отсчётом
- Пауза
- Навигация по вопросам
- Результаты с процентом

### Админ-панель
- Управление заданиями
- Модерация контента
- Статистика платформы
